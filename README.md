# WasteWise

## Overview

WasteWise is an ongoing research project focused on developing an Explainable AI-based Decision Support Framework for Urban Waste Governance.

The repository currently contains the initial research, problem analysis, and reference material used to understand the challenges in municipal solid waste management.
